# ArvyaX - Immersive Nature Journaling & Mood Analysis

ArvyaX is a mindfulness platform where users complete nature immersion sessions and record their thoughts. This system uses AI to analyze emotions and provide long-term mental health insights.

## 🚀 Features
- **Nature Journaling:** Log entries for Forest, Ocean, and Mountain sessions.
- **AI Emotion Analysis:** Real-time sentiment and mood extraction using Gemini/OpenAI.
- **Insights Dashboard:** Visualize top emotions and trends over time.

## 🛠️ Tech Stack
- **Frontend:** React / Next.js
- **Backend:** Node.js + Express
- **Database:** [Your DB - e.g., MongoDB/PostgreSQL]
- **AI:** Google Gemini API

## 🏁 Getting Started
1. Clone the repo: `git clone [your-repo-link]`
2. Install dependencies: `npm install` (in both /frontend and /backend)
3. Add your `.env` keys (API_KEY, DATABASE_URL).
4. Run the app: `npm start`
